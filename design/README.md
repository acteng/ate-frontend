# Design files

This directory contains the source design files used by ATE Frontend.

## GOV.UK Frontend

The [large icons](https://github.com/alphagov/govuk-frontend/blob/main/packages/govuk-frontend/src/govuk/assets/images/govuk-icon-512.png)
provided by GOV.UK Frontend are created manually. This makes it difficult to change the background colour for other
departments.

In attempt to source an SVG for these icons, [Matt Eason](https://github.com/matteason) recreated
[govuk-icon.svg](govuk-frontend/govuk-icon.svg) based on the highest fidelity logo [Crown_Black.ai](https://github.com/alphagov/govuk-brand-assets/blob/main/Logo/Crown/Black/Crown_Black.ai)
with the same padding as the existing icons.
